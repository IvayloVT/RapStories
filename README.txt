Rap Stories Full Vault System

- index.html
- artist-archive.html
- category.html
- timeline.html
- vault-data.js
- vault-system.js

How it works:
- Home page routes users into the archive through live vault search.
- Archive page supports search, tag filters, era filters, category filters, and sorting.
- Category page builds dynamic category hubs from the shared data file.
- Timeline page builds connected artist timeline entries from the same data source.
